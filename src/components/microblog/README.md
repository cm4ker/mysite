# Компонент Microblog

Компонент для отображения микроблога с фейковыми записями.

## Возможности

- **Список постов**: Отображение всех постов в виде сетки карточек
- **Детальный просмотр**: Полный текст поста при клике на карточку
- **Метаинформация**: Дата публикации, теги, количество лайков и просмотров
- **Адаптивный дизайн**: Корректное отображение на всех устройствах
- **Анимации**: Плавные переходы между списком и детальным просмотром

## Структура поста

```typescript
interface MicroblogPost {
  id: number;
  date: string;           // Дата в формате YYYY-MM-DD
  title: string;          // Заголовок поста
  preview: string;        // Короткое описание для карточки
  content: string;        // Полный текст поста
  tags: string[];         // Массив тегов
  likes: number;          // Количество лайков
  views: number;          // Количество просмотров
}
```

## Использование

```tsx
import Microblog from './components/microblog';

function App() {
  return (
    <div>
      <Microblog />
    </div>
  );
}
```

## Навигация

Компонент добавлен в систему навигации:
- Горячая клавиша: `M` - переход к микроблогу
- Command Palette: `Ctrl+K` → "Микроблог"
- Секция ID: `#microblog`

## Стили

Компонент использует CSS переменные темы:
- `--text`: Основной цвет текста
- `--text-secondary`: Второстепенный цвет текста
- `--bg-primary`: Основной фон
- `--bg-secondary`: Второстепенный фон
- `--accent`: Акцентный цвет
- `--border`: Цвет границ
